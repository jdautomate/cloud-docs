Activate a User account with the OpenVPN Connect app

Activate a User account with the CloudConnexa Connect app

Prerequisites: Each User must download the CloudConnexa Connect app and get a CloudConnexa Profile.

If the CloudConnexa Connect application has already been pushed to Users with the Device management system, those Users receive an email with a temporary password. Each User needs to import a Profile to connect to CloudConnexa, and can directly import a Profile in the Connect app with the temporary password.

Here is an example of the information that is sent in the User email:

  • Profile import link: https://technop.openvpn.cloud

  • Username: clientp

  • Temporary Password: xxxxxxxxxxxx

Each User must follow these steps to activate an account with the CloudConnexa Connect app:

  1. Open the CloudConnexa Connect app, sign in with the temporary password, and click Next.

  2. Set a new password, confirm it, review and agree to the terms and conditions, and click Confirm.

  3. Select a CloudConnexa region.
