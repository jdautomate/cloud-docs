Cyber Shield Domain Filtering Category Groups and Categories

Overview

This document lists the eight domain-filtering content category groups and the related content categories within each category group. The category groups are:

  • Malicious

  • Adult Content

  • Aggressive Content

  • Alcohol / Tobacco / Illegal Drugs

  • Hacking and Cracking

  • Bandwidth Consuming

  • Gambling and Games

  • Time Stealer

For each category group, information is provided in a table as to whether that category group will be blocked when one of the preset modes (Basic, Safe Browsing, High Productivity) is chosen. Cyber Shield Domain Filtering also offers a Custom mode that allows granular selection of any category from any Category Group.

For each content category, a description of the kind of content that falls into that category is provided along with a test domain. The procedure to test is the following:

  1. Access the Cyber Shield page from the Administration portal and on the Domain Filtering pane turn Monitoring ON. This enables you to monitor domain traffic.

  2. After monitoring is activated and if you want to block content categories, click on the edit (pencil) icon. The Domain Filtering window is displayed.

  3. Click the Save button to save the changes and click X to exit.

  4. Confirm that you are connected to the WPC using the same profile from the configured account.

  5. Open a separate incognito browser tab and enter the OpenVPN test domain for the category you selected into the address bar. For example, to test Malware, enter http://malware-test-domain.openvpn.com/ and hit enter.

  6. Confirm from the Domain Filtering graph that the number of blocked domains has increased by 1.

    Note

    Domain blocking depends on DNS records' TTL timeout and may take up to 24 hours to take effect.

Category: Adware

Websites that distribute Potentially Unwanted Programs (PUP) or Potentially Unwanted Applications (PUA) and ads via Pop-Ups. PUP and PUA are typically distributed within 3rd party bundled software packages when a user installs other desired software. PUP or PUA are not innately malicious, but can impact device performance, or pose a security risk by installing root certificates that can be exploited by malicious actors.

Caution

Does NOT include advertisement websites.

These websites may be accessed by links while visiting News/Media, or Social Networking websites. To ensure added blocking, enable those categories also.

Why Should I Choose this Category?

  • Avoid annoyance or distraction to your users

  • Increase user productivity

  • Protect user data

Category: Command and Control

Domains used for Command and Control of malware bots installed on user's computer. Also known as C2 or C&amp;C – Command and Control establishes an unauthorized connection to a user’s computer.

This usually happens when a user opens a phishing email or other infected resource, allowing the malicious actor to commandeer the computer and execute their own code. The malicious actor may also have access to the user’s data.

These domains may be accessed by links while visiting News/Media websites, using Webmail, or Phishing attempts. To ensure added blocking, enable those categories also.

Some negative effects of Command and Control:

  • Compromised user and customer data

Why Should I Choose this Category?

  • Prevent unauthorized access to your device or network

  • Protect user or customer data

Category: Cryptojacking

Websites used for Cryptojacking – unauthorized use of another user’s computer resources to mine cryptocurrencies. Cryptojacking is a class of malware that is distributed by phishing emails, adware, or infected websites.

Some negative effects of cryptojacking include:

  • Performance degradation

  • For Enterprise-level organizations, increased data costs

Why Should I Choose this Category?

  • Improve your security posture

  • Prevent unauthorized access to your device or network

  • Reduce potentially elevated data costs

Category: DGA

Random domains are generated using the DGA method to escape detection and establish coordinated attacks. DGA stands for Digital Generating Algorithm. This malware cyber attack method generates fictitious domains to establish communication with a Command and Control server to control the device. The fictitious command servers are decommissioned and re-generated at new IP addresses to continue coordinated attacks.

Why Should I Choose this Category?

  • Improve your security posture

  • Prevent unauthorized access to your device or network

  • Protect data

Category: Malware

Websites related to malware distribution and communication. Malware is software designed to gain unauthorized access to a user’s computer. Some websites contain forums discussing methods of distribution. Some websites are specifically created to deliver malware. Some malware distribution methods are:

  • Social Network spam – unsolicited messaging from social networks

    • To enable blocking of Social Networking, enable that category also

  • Drive by downloads from infected websites – unintentional downloads that install malware

    • To block potentially infected websites, enable the Web spam or scam category also

  • Exploit kits – A Remote Access Tool used to exploit a user’s computer

    • To block access to potentially dangerous sites, enable the Command and Control and Phishing categories also

Why Should I Choose this Category?

  • Improve your security posture

  • Prevent unauthorized access to your device or network

  • Protect data

Category: Ransomware

Websites related to ransomware distribution and control. Ransomware is malicious software and techniques used to encrypt or otherwise make critical data and systems inaccessible. Users are manipulated to submit remittances to regain access to data. Ransomware can be distributed in the form of popups that require action from the user to avoid data loss, performance degradation, unauthorized distribution, or from email.

To ensure added protection against ransomware, you may need to enable the Webmail, File storage, Warez (includes torrents), Cryptocurrency and mining categories also.

Why Should I Choose this Category?

  • Improve your security posture

  • Protect data

  • Protect your company’s reputation and financial resources

Category: Web spam or scam

Websites that advertise fake services, products and other fraudulent schemes. These sites distribute unauthorized spam, or solicit fraudulent schemes. Web spam is evasive and is usually distributed from zombie computers that act as proxies to deliver unauthorized emails, tricking the users into clicking malicious links that may contain fake services, products, or fraudulent schemes. Less malicious spam can be a distraction for your users.
